# Language Syntax

## Operators

### End of Line Operator

`~`

### Code Block Operators

`<>`

### Subscript Operator

`{}`

### Assignment Operator

`:>`

### Arithmetic Operators

**Addition:** `(+)`

**Substraction:** `(-)`

**Multiplication:** `(*)`

**Division:** `(/)`

### Logical Operators

**AND:** `(&&)` or `(AND)`

**OR:** `(||)` or `(OR)`
